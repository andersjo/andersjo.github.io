Anders Johannsen

I recently joined the Copenhagen Applied Statistical Learning (COASTAL) group at the University of Copenhagen as a post.doc. My research focuses on developing robust language technology that allows us to cross domain and language barriers.

Software

  • Rungsted, a very fast and easily hackable structured perceptron implementation in Python. Supports cost-sensitive updates, constrained inference, and feature hashing.
  • FrameNet annotation tool. A browser interface for marking up text with frame semantic information. The GUI displays textual descriptions of frame and arguments as needed, alleviating the need to memorize all the definitions.
  • Andreord.dk. Semantic networks presented in a graphical and accessible manner. Available as a public web site here and in a version with cross-language links there.
